## 作業内容

### 1. カラーパレット適用（Tailwind設定）

**`tailwind.config.js`** にカスタムカラーを追加:

```js
theme: {
    extend: {
        colors: {
            primary:  '#2563EB',
            success:  '#16A34A',
            danger:   '#DC2626',
            subBg:    '#F3F4F6',
            appText:  '#111827',
        },
    },
},
```

---

### 2. 共通レイアウト（`layouts/app.blade.php`）

- サイドバー幅・フォントサイズ見直し
- アクティブメニューのハイライト（現在のルートに応じて）
- ボタン最低44px確保
- 横画面固定（`<meta>`タグ追加）

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, orientation=landscape">
```

---

### 3. 患者一覧画面（`customers/index.blade.php`）

要件定義14.5に基づく:

- **上部**: 検索バー（患者ID / 院 / 治療内容 / 治療部位）
- **下部**: テーブル形式の患者リスト
- 行タップで患者詳細へ遷移
- テーブル行の最低高さ44px

```
┌─────────────────────────────────────┐
│ 🔍 患者ID  院  治療内容  治療部位  [検索] │
├──────┬──────┬──────┬──────────────┤
│ 患者ID │ 氏名 │ 院  │ 最終診療日   │
├──────┼──────┼──────┼──────────────┤
│ P001  │ 山田  │ 渋谷 │ 2026/03/20   │
└──────┴──────┴──────┴──────────────┘
```

---

### 4. 患者詳細画面（`customers/show.blade.php`）

要件定義14.5に基づく（最重要画面）:

- **左カラム（1/3）**: 基本情報カード
- **右カラム（2/3）**: 診療履歴タイムライン

タイムライン表示:
```
│
●── 2026/03/20  根管治療（神経治療）
│   治療部位: 右上7番  担当: 佐藤先生
│
●── 2026/03/10  歯石除去（スケーリング）
│   治療部位: 全顎  担当: 田中先生
```

表示項目:
- 日時
- 治療内容
- 治療部位（NULLの場合は非表示）
- 担当医
- ステータスバッジ（予約:青 / 完了:緑 / キャンセル:グレー）

---

### 5. ステータスバッジ（`components/badge.blade.php`）

要件定義14.7に基づく:

| ステータス | 色 |
|---|---|
| reserved（予約） | 青 `bg-blue-100 text-blue-800` |
| completed（処置完了） | 緑 `bg-green-100 text-green-800` |
| cancelled（キャンセル） | グレー `bg-gray-100 text-gray-600` |
| エラー | 赤 `bg-red-100 text-red-800` |

---

### 6. CSVインポート画面（`import/index.blade.php`）

要件定義14.5に基づく:

- シンプルなファイル選択UI
- インポート実行ボタン（44px以上）
- 結果表示（成功件数 / エラー件数）

---

### 7. タブレット対応の横断的確認

全画面で以下を確認:

- [ ] ボタン・タップ領域が最低44px
- [ ] 横画面（1024px以上）で崩れない
- [ ] フォントサイズが小さすぎない（最低14px）
- [ ] 不要なアニメーションがない

---

```bash
npm run build
```
